# infraguard config

Manage InfraGuard configuration.

## Subcommands

### set

Set a configuration value:
```bash
infraguard config set lang zh
```

### get

Get a configuration value:
```bash
infraguard config get lang
```

### list

List all configuration values:
```bash
infraguard config list
```

### unset

Remove a configuration value:
```bash
infraguard config unset lang
```

## Available Settings

- `lang`: Output language (`en` or `zh`)

For more details, see [Configuration](../user-guide/configuration).
