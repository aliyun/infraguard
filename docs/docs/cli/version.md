# infraguard version

Print version information for InfraGuard CLI and OPA.

## Synopsis

```bash
infraguard version [flags]
```

## Description

The `version` command displays version information for:
- InfraGuard CLI version
- OPA (Open Policy Agent) version used by InfraGuard

## Examples

### Display Version Information

```bash
infraguard version
```

Output:
```
InfraGuard: 0.5.0
OPA: 1.12.1
```

## Exit Codes

- `0`: Success
