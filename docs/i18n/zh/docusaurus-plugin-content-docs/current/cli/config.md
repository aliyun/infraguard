# infraguard config

管理 InfraGuard 配置。

## 子命令

### set

设置配置值：
```bash
infraguard config set lang zh
```

### get

获取配置值：
```bash
infraguard config get lang
```

### list

列出所有配置值：
```bash
infraguard config list
```

### unset

删除配置值：
```bash
infraguard config unset lang
```

## 可用设置

- `lang`: 输出语言（`en`、`zh`、`es`、`fr`、`de`、`ja` 或 `pt`）

有关更多详细信息，请参阅[配置](../user-guide/configuration)。
