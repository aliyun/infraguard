# infraguard version

打印 InfraGuard CLI 和 OPA 的版本信息。

## 概要

```bash
infraguard version [选项]
```

## 说明

`version` 命令显示以下版本信息：
- InfraGuard CLI 版本
- InfraGuard 使用的 OPA（Open Policy Agent）版本

## 示例

### 显示版本信息

```bash
infraguard version
```

输出：
```
InfraGuard: 0.5.0
OPA: 1.12.1
```

## 退出代码

- `0`: 成功
