# infraguard version

Imprimir informações de versão para InfraGuard CLI e OPA.

## Sinopse

```bash
infraguard version [flags]
```

## Descrição

O comando `version` exibe informações de versão para:
- Versão do InfraGuard CLI
- Versão do OPA (Open Policy Agent) usada pelo InfraGuard

## Exemplos

### Exibir Informações de Versão

```bash
infraguard version
```

Saída:
```
InfraGuard: 0.5.0
OPA: 1.12.1
```

## Códigos de Saída

- `0`: Sucesso
