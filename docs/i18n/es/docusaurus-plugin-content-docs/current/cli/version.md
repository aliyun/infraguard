# infraguard version

Mostrar información de versión para InfraGuard CLI y OPA.

## Sinopsis

```bash
infraguard version [flags]
```

## Descripción

El comando `version` muestra información de versión para:
- Versión de InfraGuard CLI
- Versión de OPA (Open Policy Agent) utilizada por InfraGuard

## Ejemplos

### Mostrar Información de Versión

```bash
infraguard version
```

Salida:
```
InfraGuard: 0.5.0
OPA: 1.12.1
```

## Códigos de Salida

- `0`: Éxito
