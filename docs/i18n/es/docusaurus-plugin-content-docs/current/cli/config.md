# infraguard config

Gestionar la configuración de InfraGuard.

## Subcomandos

### set

Establecer un valor de configuración:
```bash
infraguard config set lang es
```

### get

Obtener un valor de configuración:
```bash
infraguard config get lang
```

### list

Listar todos los valores de configuración:
```bash
infraguard config list
```

### unset

Eliminar un valor de configuración:
```bash
infraguard config unset lang
```

## Configuraciones Disponibles

- `lang`: Idioma de salida (`en`, `zh`, `es`, `fr`, `de`, `ja`, o `pt`)

Para más detalles, consulte [Configuración](../user-guide/configuration).
