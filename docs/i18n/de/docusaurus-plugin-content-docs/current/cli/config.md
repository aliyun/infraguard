# infraguard config

InfraGuard-Konfiguration verwalten.

## Unterbefehle

### set

Einen Konfigurationswert setzen:
```bash
infraguard config set lang de
```

### get

Einen Konfigurationswert abrufen:
```bash
infraguard config get lang
```

### list

Alle Konfigurationswerte auflisten:
```bash
infraguard config list
```

### unset

Einen Konfigurationswert entfernen:
```bash
infraguard config unset lang
```

## Verfügbare Einstellungen

- `lang`: Ausgabesprache (`en`, `zh`, `es`, `fr`, `de`, `ja`, oder `pt`)

Für weitere Details siehe [Konfiguration](../user-guide/configuration).
