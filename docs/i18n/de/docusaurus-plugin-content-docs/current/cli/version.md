# infraguard version

Versionsinformationen für InfraGuard CLI und OPA ausgeben.

## Synopsis

```bash
infraguard version [flags]
```

## Beschreibung

Der Befehl `version` zeigt Versionsinformationen für:
- InfraGuard CLI-Version
- Von InfraGuard verwendete OPA (Open Policy Agent)-Version

## Beispiele

### Versionsinformationen Anzeigen

```bash
infraguard version
```

Ausgabe:
```
InfraGuard: 0.5.0
OPA: 1.12.1
```

## Exit-Codes

- `0`: Erfolg
