# infraguard version

Afficher les informations de version pour InfraGuard CLI et OPA.

## Synopsis

```bash
infraguard version [flags]
```

## Description

La commande `version` affiche les informations de version pour :
- Version d'InfraGuard CLI
- Version d'OPA (Open Policy Agent) utilisée par InfraGuard

## Exemples

### Afficher les Informations de Version

```bash
infraguard version
```

Sortie :
```
InfraGuard: 0.5.0
OPA: 1.12.1
```

## Codes de Sortie

- `0`: Succès
