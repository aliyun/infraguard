# infraguard config

Gérer la configuration d'InfraGuard.

## Sous-commandes

### set

Définir une valeur de configuration :
```bash
infraguard config set lang fr
```

### get

Obtenir une valeur de configuration :
```bash
infraguard config get lang
```

### list

Lister toutes les valeurs de configuration :
```bash
infraguard config list
```

### unset

Supprimer une valeur de configuration :
```bash
infraguard config unset lang
```

## Paramètres Disponibles

- `lang`: Langue de sortie (`en`, `zh`, `es`, `fr`, `de`, `ja`, ou `pt`)

Pour plus de détails, consultez [Configuration](../user-guide/configuration).
